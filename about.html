About Cobalt Media Downloader

This extension was created to make downloading media easier using cobalt.tools.

Created by:
desgubernamentalizar
Version:
2.3.0
GitHub Repository:
github.com/nikoriki/cobaltextension

Features

  • Quick download of media from any supported website
  • Choose to download audio only, video only, or both (auto mode)
  • Multiple ways to download:
    • Popup interface
    • Right-click context menu
    • Keyboard shortcuts
  • Download history tracking
  • Dark theme interface
